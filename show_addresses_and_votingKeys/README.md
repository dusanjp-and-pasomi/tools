# 使用方法
- ダウンロード  
shoestringNodeの作業ディレクトリで、  
`wget https://github.com/dusanjp-and-pasomi/tools/raw/refs/heads/main/show_addresses_and_votingKeys/address.py`  
- 実行  
`python3 address.py`
